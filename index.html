Introduction

HTML stands for Hypertext MarkupLanguage
HTML is the standard markup languagefor Web pages
HTML elements laydown the basic layout of HTML pages

HTML Elements

An HTML element is a start tag and an end tag with content in between:

eg.

<h1>This is a Heading Tag</h1>

HTML Attributes

  • HTML elements can have any attributes
  • HTML attributes provides additional information about HTML webpages

HTML Structure

<HTML>
<HEAD>
<TITLE> </TITLE> </HEAD>
<BODY>
All coding and layout part done inside
</body>
</HTML>

HTML Headings

There are various heading tags in HTML:

eg:
  • <h1>This is H1 heading tag </h1>
  • <h2>This is H2 heading tag </h2>
  • <h3>This is H3 heading tag </h3>

HTML Paragraphs

HTML paragraphs are defined as:

<p>This is Paragraph Tag</p>

HTML Links

HTML Links are defined as:

<a href="#">This is an anchor Tag</a>

HTML Images

HTML Images are defined with: <img> tag

The source file (src), alternative text(alt),width,height,are provided as properties

HTML Buttons

HTML Buttons are defined as: <button> tags

<button>Click me</button>

HTML Lists

HTML lists are defined with <ul> (unordered/bullet list) or <ol> (ordered list) tags, followed by <li> tags (list items):

<ul>
  <li>Viirat</li>
  <li>Rohit</li>
 <li>Rahul</li>
</ul >
